# movingRobots
Threads example

# Required
1. ubuntu
2. QtCreator

##  Install QtCreator
Open terminal and run:

  1. sudo apt-get install build-essential

  2. sudo apt-get install qtcreator

  3. sudo apt-get install qt5-default

## Open QtCreator

  Open terminal and run:
  
  1. qtcreator
  
  
After that, select Open Project and then the *.SO file.
